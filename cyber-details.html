Project Details:

Software you'll need for this project to run before following instructions:
  • Node JS 20.13.1
  • VS Code or alternative (This guide will use VS Code)

This project is a discord bot that is run locally on your computer. Upon boot up it introduces itself in the server it's in and informs users on what you can do with it. It can provide questions, multiple choice or short answer questions, and allows you to attempt to solve it and see if you're correct or not to learn easily and quickly in the Discord UI/system, which makes it easy to do along with your friends if you want to! Additionally, it provides a huge chunk of news and daily tip upon boot-up, and every 24hrs does so again. Another thing this bot does is it listens to whatever you write in the chat always and if you write a word, like "ctf" or "capture the flag", then it'll drop a large amount of info to you privately about that topic to teach you about that very topic. The idea is so if you ask "what's ctf?" in the server, instead of anyone needing to answer, you'll receive your answer immediately. Finally, there's commands for learning resources on a wide variety (over 50) of CyberSecurity topics that make it easy to learn about a topic and sometimes be able to actually learn how to apply these concepts or prevent them.


Instructions

  • After downloading the code and extracting the zip file, pull it up in VS Code
  • Click on "New Terminal" and enter the command "npm install"
  • Enter the command "node deploy-commands.js"
  • Enter the command "node index.js"

And then you are done!



Resume points:
  • Utilizes "RapidAPI" API for fetching new news within the field from major companies such as: "cnbc", "wired", "threadpost" etc. automatically daily and returns it a dedicated section of the server
  • Implemented automated features including daily cybersecurity tips and interactive commands for resources and quizzes, enhancing user engagement and knowledge.
  • Enabled keyword-based information retrieval and personalized responses to facilitate interactive discussions within the server.
