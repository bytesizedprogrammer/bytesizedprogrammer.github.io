Project Details:

This project is a website promoting The Fuller Center for NY. It is edited and created using Strikingly.com and its in built usage of HTML/CSS option. I primarily used the built in options, but for certain things to look better, the usage of HTML and CSS was implemented.


Resume points:
  • Enhanced website design to improve user engagement and satisfaction, resulting in an 82% increase in weekly user visits
  • Revamped website layout to improve visual appeal and usability
